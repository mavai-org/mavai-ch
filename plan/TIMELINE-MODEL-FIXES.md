# Timeline Model Release Reorganization

## Overview
This document proposes a curated list of major LLM model releases for the javai.ch timeline, focusing on significant milestones rather than every incremental update. Each entry includes the company name to provide context for users unfamiliar with the models.

## Rationale

### What We Removed
- **Minor version updates**: Claude 3.5, Claude 4.5, Claude 4.6, GPT-5.4 Mini/Nano — incremental improvements that don't represent paradigm shifts
- **Early single-company research releases**: Claude 1 announced same day as GPT-4; Llama 1 (researchers-only); Google Bard (quickly superseded)
- **Marginal improvements in existing families**: Gemini 1.5 Pro, Gemini 2.0, Gemini 2.5, Mistral Large, Mistral Magistral, Grok 3
- **Non-LLM models**: Sora (text-to-video), though significant for multimodal generative AI
- **Region-specific or lower-impact releases**: Manus AI (Chinese agent, less relevant for Swiss audience), lesser-known models

### What We Kept
1. **Foundation-shifting releases**: ChatGPT (public LLM era), GPT-4, Claude 4, GPT-5, o1 (reasoning paradigm)
2. **Open-source breakthroughs**: Llama 2 (first openly available commercial model), Llama 4 MoE (first open multimodal MoE), DeepSeek-R1 (competitive non-Western reasoning)
3. **Company diversification**: Added geographical and corporate diversity (Meta, Google DeepMind, Anthropic, Mistral, DeepSeek, Microsoft, Apple)
4. **Major integration moments**: OS-level adoption (Windows, iOS, macOS) showing LLMs moving from niche to mainstream infrastructure
5. **Capability paradigm shifts**: o1/o3 reasoning models, computer use/agents, Claude Mythos Preview (cybersecurity), demonstrating evolution beyond text generation toward specialized high-impact capabilities

## Implementation Notes

- All headlines now include **company name** for clarity (e.g., "OpenAI releases GPT-5", "Apple integrates ChatGPT")
- Each entry includes **why it matters** for Swiss regulatory and business context
- Open-source models (Llama, DeepSeek) are highlighted to show ecosystem diversity
- Timeline emphasizes **adoption moments** (Apple, Microsoft, ChatGPT launch) alongside pure capability releases
